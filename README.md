# Germany Job Portals Directory

A curated, visually stunning gateway to career opportunities in Germany.

## Features Added
- **Premium Design Aesthetics**: Upgraded UI using modern CSS variables, glassmorphism, and a dynamic layered background.
- **Search Functionality**: A responsive, real-time JavaScript-driven search bar that filters links by both their visible titles and hidden meta-tags (so searching "medical" surfaces all health-related boards).
- **Categorization Context**: Each domain link features categorical emojis and meta tags allowing you to quickly spot what kind of sector they cater to (IT, Medical, Startups, General, Regional).
- **Responsive Layout**: Utilizing CSS Grid, the interface easily scales down beautifully for mobile browsers.
- **Dark-Theme Footer**: Persistent interactive footer with a live-updating clock and copyright date.

## Tech Stack
- HTML5
- CSS3 (Vanilla, CSS Grid, Glassmorphism, Micro-animations)
- Vanilla JavaScript 

> Built to make job hunting a little more delightful.
